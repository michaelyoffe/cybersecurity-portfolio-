# Technical Labs

This directory contains hands-on laboratory exercises focused on technical security controls, data forensics, and system hardening. Each lab includes a detailed breakdown of the tools used and the security objectives achieved.

## Labs Included

| Lab Name | Primary Tools | Objective |
| :--- | :--- | :--- |
| [SQL Query Filtering](./sql-filter-queries.md) | SQL, MariaDB | Investigate security threats by filtering login logs and employee data. |
| [Linux File Permissions](./linux-lab.md) | Bash, Linux CLI | Harden system security by auditing and modifying user access levels. |

---
## Skills Demonstrated
* **Database Security:** Writing efficient SQL queries to identify unauthorized access.
* **System Administration:** Managing Linux users, groups, and permissions (chmod/chown).
* **Data Forensics:** Analyzing log files to reconstruct security incidents.
